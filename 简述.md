# NovelAgent

基于 **LangChain** + **LangGraph** 的 AI 长篇小说写作辅助工具。

## 架构

```
后端（FastAPI + LangChain）←→ 前端（React + Vite + Zustand）
        │
    ┌───┴───┐
    │       │
  JSON     ChromaDB
 文件存储   (RAG向量)
```

## 核心流程

### 大纲驱动
卷 → 章 → 节 三级树形大纲，用户设计结构，AI 不修改大纲。

### LangChain Agent
单 Agent 通过 LangGraph `create_react_agent` 实现 ReAct 循环：

```
用户指令 → LLM思考 → 工具调用 → 观察结果 → LLM思考 → ... → finish
```

### 可用工具（10个）
| 工具 | 用途 |
|------|------|
| `get_outline` | 读取大纲结构 |
| `get_characters` | 读取角色档案 |
| `get_world_settings` | 读取世界观设定 |
| `get_summaries` | 读取前文摘要 |
| `search_memory` | RAG 向量检索已写内容 |
| `write_content` | 创作正文 |
| `polish_content` | 润色正文 |
| `review_content` | 审查正文质量 |
| `summarize_content` | 生成结构化摘要 |
| `finish` | 完成任务返回结果 |

### RAG 记忆
- ChromaDB + sentence-transformers 向量存储
- 保存时自动嵌入（内容 > 50 字）
- Agent 通过 `search_memory` 工具检索相关前文

## 快速开始

```bash
pip install -r backend/requirements.txt
cd frontend && npm install
start.bat
```

- 后端: `http://localhost:8001`
- 前端: `http://localhost:3001`
- API 文档: `http://localhost:8001/docs`
